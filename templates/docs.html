Getting started

Four steps from invite to your first song.

01

Invite Maya

Use the invite link to add Maya to your server with the permissions it needs to join voice channels and manage the queue.

02

Set your prefix

Run setprefix (alias prefix) to change the command prefix for your server, or skip it to keep the default.

03

Join a voice channel

Hop into a voice channel and run play with a song name or link. Maya joins automatically and starts the queue.

04

Or use the dashboard

Prefer clicking over typing? Log in to the dashboard to control playback, queue, and volume from your browser.

FAQ

How do I vote for Maya?
Use the Vote link in the nav bar — it takes a few seconds and helps other people find Maya.
Is Maya free to use?
Yes — every command and the dashboard are free, with no paywalled features.
What does 24/7 mode do?
Run 247 to keep Maya connected to a voice channel even when it's alone, so the queue survives quiet moments instead of disconnecting.
Can I change the command prefix?
Yes — run setprefix (or prefix) followed by whatever you'd like to use instead.
Where do I get support?
Run support in any server with Maya to get an invite to the support server.
