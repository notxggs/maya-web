🎵 Music commands

13 commands — playback, queue, and volume control.

play
↳ Searches for a song by name or takes a direct link (YouTube, Spotify, SoundCloud, etc.) and adds it to the queue. If Maya isn't already in a voice channel, she joins yours automatically and starts playing right away if nothing else is queued.
↳ Aliases: p
pause
↳ Pauses the current track in place without clearing the queue or disconnecting. Playback picks up exactly where it left off with resume.
↳ Aliases: none
resume
↳ Resumes a paused track from the exact position it was paused at.
↳ Aliases: unpause
skip
↳ Immediately stops the current track and moves on to the next song in the queue. If the queue is empty, playback stops (unless autoplay is on).
↳ Aliases: s
previous
↳ Jumps back and replays the track that was playing right before the current one, pulling it from Maya's recent-track history rather than the upcoming queue.
↳ Aliases: prevback
replay
↳ Restarts the currently playing track from 0:00 without touching the rest of the queue — handy if you missed the start of a song.
↳ Aliases: none
stop
↳ Clears the entire queue, stops playback, and disconnects Maya from the voice channel. This is a full reset — use pause instead if you just want to take a break.
↳ Aliases: leavedcdisconnect
queue
↳ Displays the list of upcoming tracks in order, showing up to 25 at a time along with who requested each one.
↳ Aliases: q
nowplaying
↳ Shows details on the track currently playing — title, artist, and a progress bar showing elapsed vs. total duration.
↳ Aliases: np
shuffle
↳ Randomizes the order of every track currently sitting in the queue. Doesn't affect the track that's already playing.
↳ Aliases: none
loop
↳ Toggles repeat mode on the current track — once enabled, the track restarts automatically instead of moving to the next one when it ends. Run it again to turn it off.
↳ Aliases: repeat
autoplay
↳ When enabled, Maya automatically queues a related track once the queue runs empty, so the music keeps going without anyone needing to add more songs.
↳ Aliases: ap
volume
↳ Views the current playback volume, or sets a new one (0–150) when given a number. Values above 100 boost beyond normal loudness — useful for quiet sources, but can introduce distortion at the high end.
↳ Aliases: vol

⚙ Config commands

2 commands — server-level settings.

setprefix
↳ Changes the prefix used to trigger Maya's commands in this server (up to 5 characters). Requires Manage Server permission.
↳ Aliases: prefix
247
↳ Toggles 24/7 mode. When on, Maya stays connected to the voice channel even after everyone else leaves, so the queue is still there when people come back instead of resetting. Requires Manage Server permission.
↳ Aliases: none

ℹ Information commands

6 commands — help, stats, and support.

help
↳ Shows the full list of commands grouped by category. Pass a specific command name (e.g. help play) to see its detailed usage and aliases.
↳ Aliases: h
invite
↳ Sends the official invite link to add Maya to one of your own servers.
↳ Aliases: none
membercount
↳ Shows how many members are currently in the server.
↳ Aliases: mc
ping
↳ Checks Maya's latency to Discord's servers, useful for confirming she's responsive and diagnosing lag.
↳ Aliases: none
stats
↳ Shows detailed statistics about Maya herself — uptime, server count, memory usage, and current load.
↳ Aliases: botinfobi
support
↳ Sends an invite link to Maya's official support server, where you can ask questions or report issues.
↳ Aliases: none

🛠 Utility commands

6 commands — avatars, banners, and server info.

avatar
↳ Displays a user's Discord avatar at full resolution. Defaults to your own avatar if no user is mentioned.
↳ Aliases: avpfp
banner
↳ Displays a user's profile banner, if they have one set on their Discord account.
↳ Aliases: none
serverbanner
↳ Displays the server's banner image, if the server has one set (requires a server boost level that supports banners).
↳ Aliases: sbanner
servericon
↳ Displays the server's icon at full resolution.
↳ Aliases: sicon
serverinfo
↳ Shows detailed information about the server — creation date, owner, member count, boost level, and more.
↳ Aliases: si
userinfo
↳ Shows detailed information about a user — account creation date, server join date, and roles. Defaults to yourself if no user is mentioned.
↳ Aliases: uiwhois

⚡ No-Prefix commands

1 command — skip typing the prefix entirely.

noprefix
↳ Lets you run Maya's commands by typing just the command name, without needing to type the server's prefix first.
↳ Aliases: none
